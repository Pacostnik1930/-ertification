cat > Домашние\ животные

cat > Вьючные\ животные


Чтобы объединить содержимое двух файлов, используем команду cat с символом перенаправления >>:


cat Домашние\ животные >> Вьючные\ животные


Теперь в файле "Вьючные животные" содержится информация из обоих файлов.

Просмотреть содержимое файла можно с помощью команды cat:


cat Вьючные\ животные


Чтобы переименовать файл, используем команду mv:


mv Вьючные\ животные Друзья\ человека


Теперь файл "Вьючные животные" переименован в "Друзья человека".

REATE TABLE молодые_животные AS

     SELECT 
         имя,
         команда,
         дата_рождения,
         TIMESTAMPDIFF(MONTH, дата_рождения, CURDATE()) AS возраст_в_месяцах
     FROM
         (SELECT 
             имя,
             команда,
             дата_рождения
         FROM
             (SELECT 
                 имя,
                 команда,
                 дата_рождения
             FROM
                 лошади
             UNION ALL SELECT 
                 имя,
                 команда,
                 дата_рождения
             FROM
                 ослы) AS животные
         WHERE
             дата_рождения > DATE_SUB(CURDATE(), INTERVAL 3 YEAR)
                 AND дата_рождения < DATE_SUB(CURDATE(), INTERVAL 1 YEAR)) AS молодые_животные;

     SELECT 'Лошади' AS источник, имя, команда, дата_рождения FROM лошади

     SELECT 'Ослы' AS источник, имя, команда, дата_рождения FROM ослы;
